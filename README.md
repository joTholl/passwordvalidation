# Dieses Programm dient dazu Passwörter auf ihre Sicherheit zu überprüfen

## Es werden folgende Kriterien überprüft:

### - 3 der folgenden 4 Kriterien müssen erfüllt sein:

- mindestens eine Ziffer
- mindestens ein Großbuchstabe
- mindestens ein Kleinbuchstabe
- mindestens ein Sonderzeichen

##### - mindestens 8 Zeichen
##### - nicht in einer Liste mit häufigen/schwachen Passwörtern

##### Es wurde auf Regex verzichtet, um Schleifen und Arrays zu üben
